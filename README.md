# mini-compilateur-JS
Dans ce projet, nous avons conçu et implémenté un compilateur simplifié pour un
sous-ensemble du langage JavaScript. L’objectif principal est de comprendre en profon
deur les mécanismes internes d’un compilateur, notamment l’analyse lexicale et l’analyse
syntaxique.
Le projet s’inscrit dans une approche pédagogique visant à maîtriser :
— Latransformation d’un flux de caractères en tokens
— Lavalidation syntaxique à l’aide d’une grammaire formelle
— Lagestion des erreurs
— Lavisualisation des résultats via une interface graphique
Ce travail permet également d’introduire des concepts avancés tels que les grammaires
non contextuelles, les analyseurs LL(1), et la construction d’arbres syntaxiques
